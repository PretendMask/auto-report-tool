## 📖 项目介绍

在日常工作中，员工报销通常需要提交订单截图、发票等文件。手动整理这些报销单非常繁琐，容易出错且效率低下。

**本工具可以实现：**
- 自动扫描指定文件夹中的报销相关文件
- 自动提取文件名中的日期、事由、金额等信息
- 自动生成格式统一的 Excel 报销表格
- 自动将订单图片和发票（PDF 转图片）插入到 Excel 中
- 显示处理进度和当前处理文件，提供良好可视化反馈
  
**缺点：**
- 需要手动命名电脑的订单图片和发票名称

## ✨ 主要功能

- 支持**双击直接运行**（无需安装 Python 环境）
- 一键选择报销文件所在文件夹即可开始处理
- 自动生成美观统一的 Excel 报销汇总表
- 支持图片自动插入（订单图 + 发票图）
- PDF 发票自动转换为图片并插入

---

## 📌 使用方法

### 1. 双击运行
直接双击 `main.py`（或打包后的 exe）即可启动工具。

### 2. 选择文件夹
在弹出的窗口中选择包含报销文件的文件夹，然后点击开始处理。

### 3. 文件命名规范（非常重要！）

**订单图片** 和 **发票 PDF** 必须按照以下格式命名：

**格式：** `日期-事由-金额-类型`

**示例：**
- `10.30日-招待费-113-订单.jpg`
- `10.30日-招待费-113-发票.pdf`
- `11.05日-办公用品-45.5-订单.png`
- `11.05日-办公用品-45.5-发票.pdf`
